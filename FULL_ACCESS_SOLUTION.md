## 推荐访问方法：SSH隧道

SSH隧道是当前环境下最可靠、最安全的访问方式。

### 方法1：本地端口转发
在您的本地计算机上执行以下命令：

```bash
ssh -L 8080:localhost:8080 your_username@43.167.222.200
```

连接成功后，在本地浏览器访问：
- Web界面：http://localhost:8080
- API状态：http://localhost:8080/api/status

### 方法2：后台SSH隧道
如果您希望SSH隧道在后台运行：

```bash
# 创建后台SSH隧道
ssh -f -N -L 8080:localhost:8080 your_username@43.167.222.200
```

### 方法3：使用不同本地端口
如果您本地8080端口已被占用：

```bash
ssh -L 8888:localhost:8080 your_username@43.167.222.200
```

然后在本地浏览器访问 http://localhost:8888

## 服务器配置检查

服务器上的服务已正确配置：

## 需要服务器管理员协助的情况

如果以上SSH隧道方法仍然无法工作，请检查：

1. 您的本地网络是否阻止了SSH连接
2. 您的SSH密钥或密码是否正确
3. 服务器的SSH服务是否正常运行

1. 将域名解析到服务器IP (43.167.222.200)
2. 联系服务器提供商开放相应端口
